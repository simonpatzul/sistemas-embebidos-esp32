# Ejemplos de firmware – ESP32 (ESP-IDF, C)

Repositorio con **ejemplos de firmware en C** para **ESP32** usando **ESP-IDF**.

En este repositorio encontrarás ejemplos trabajados en clase para el manejo de diferentes periféricos del ESP32 vistos en clase. Cada carpeta contiene todos los archivos .h y .c de cada ejemplo, así como una breve explicación del mismo.
