# Sistemas Embebidos - ESP32 (IDF)

Repositorio del curso de Sistemas Embebidos (7mo semestre). Contiene ejemplos de ESP-IDF para ESP32, laboratorios y presentaciones de clase.

## Estructura

```
esp32-examples-espidf-main/   # Ejemplos de ESP-IDF para ESP32
  ESP32_BLE_NUS/              # Bluetooth Low Energy (Nordic UART Service)
  ESP32_BT_SPP/               # Bluetooth Classic (Serial Port Profile)
  ESP32_DAC_EJ1-3/            # Ejemplos DAC
  ESP32_GPIO_EJ1-3/           # Ejemplos GPIO
  ESP32_I2C_EJ1/              # Ejemplo I2C
  ESP32_PWM_ADC_EJ1/          # Ejemplo PWM y ADC
  ESP32_SPI_EJ1/              # Ejemplo SPI
  ESP32_UART_EJ1-2/           # Ejemplos UART

Laboratorios/                 # Guías de laboratorio (PDF)
Presentaciones/               # Diapositivas de clase (PDF, Clases 1-15)
```

## Temas cubiertos

- GPIO, UART, SPI, I2C
- DAC, PWM, ADC
- Bluetooth Classic (SPP) y BLE (NUS)
- Framework: ESP-IDF (no Arduino)
